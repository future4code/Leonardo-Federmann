Website de loja - curso de programação Web Full Stack Labenu

No curso de programação Web Full Stack da Labenu, foram desempenhadas diversas atividades aspirando ao aprendizado de ferramentas do desenvolvimento de software, dentre as quais se encontra a criação do website de uma loja própria de temática livre. Tal página incluiu a elaboração de tabelas, listas, hiperlinks, efeitos visuais e variadas outros recursos do HTML e do CSS. Algumas das tecnologias utilizadas foram:

  • HTML: uso de tags para a criação de:
    • Tabelas;
    • Hiperlinks;
    • Inserção de imagens;
    • Formulários;
    • Dentre outros...
  • CSS: utilização de seletores e propriedades de estilização visando, dentre outros:
    • Coloração;
    • Espaçamento;
    • Efeitos visuais;
    • Definição de tamanho de elementos.
